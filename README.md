体調・既往歴まとめシート

診察時に「症状」「既往歴」「服薬情報」「検査結果」などを整理し、医師へ分かりやすく伝えるためのWebアプリです。

⸻

概要

病院やクリニックで診察を受ける際、

* いつから症状があるのか
* どのような経過をたどったのか
* どんな薬を飲んでいるのか
* どんな病気にかかったことがあるのか

を正確に伝えることは意外と大変です。

本アプリは、診察前に情報を整理し、医師へ伝えやすくすることを目的として開発しました。

⸻

主な機能

基本情報整理

* 氏名
* 年齢
* 受診日
* 受診科

症状整理

* 主症状選択
* 症状経過記録
* 自由記載

既往歴管理

* 過去の病気
* 手術歴
* 入院歴

服薬情報

* 現在服用中の薬
* 過去の薬
* 自由記載

検査結果管理

* 血圧
* 血糖値
* HbA1c
* その他検査結果

診察サポート

* 医師へ相談したい内容の整理
* 印刷用シート作成
* AI要約用テキスト出力

⸻

利用方法

初回利用

1. 必要項目を入力
2. 内容を確認
3. 印刷または保存

診察時

* スマホ画面を医師へ見せる
* 印刷して医師へ渡す
* AI要約テキストをChatGPT等へ貼り付ける

⸻

対象者

* 高血圧治療中の方
* 糖尿病治療中の方
* 心療内科・精神科通院中の方
* 慢性疾患のある方
* 高齢者
* ご家族の受診をサポートする方
* 医療機関へ説明するのが苦手な方

注意事項

本アプリは診察補助および情報整理を目的としたツールです。

診断・治療・医療判断を行うものではありません。

強い胸痛、息苦しさ、麻痺、ろれつ障害、激しい頭痛、高度な血圧上昇など緊急性の高い症状がある場合は、速やかに医療機関へ相談してください。
